# Base Tokens

## Summary

Base tokens are normal tokens that do not represent a DeFi position. Base tokens are underlying token of a [DeFi Position Token](./defiTokens.mdx) and can be used to create DeFi positions.

## Endpoints

### GET [/api/v1/baseTokens](https://enso-scrape.s3.us-east-2.amazonaws.com/output/backend/baseTokens.json)
