## Endpoints

Learn more about the different endpoints of the Metadata API:

Yield opportunities:

- GET /v1/opportunities/tokenIn/ returns yield opportunities for a given token
- GET /v1/opportunities/- returns yield opportunities a given wallet address

Enso:

- GET /v1/tokens: returns all of the tokens supported by the api
- GET /v1/positions: returns data about all of the tokens supported by the api
- GET /v1/actions: returns data about all of the actions supported by the api, used to get the data needed to use the bundle api
- GET /v1/standards: returns all of the protocols supported by the api
- GET /v1/wallet: return enso wallet details
