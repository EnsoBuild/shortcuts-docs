## `GET /v1/opportunities/{address}`

Returns Yield opportunities for a wallet.
Based on the wallet content, it will return a list of opportunities.

### Request

#### Query Parameters:

| Query Param | Description                                    | Example                    |
| ----------- | ---------------------------------------------- | -------------------------- |
| `address`   | Address of the wallet to get opportunities for | address=0x6b175474e8909... |

It will return a list of action that can be used with the [bundle endpoint](/bundler-api)

  -H 'accept: application/json'
