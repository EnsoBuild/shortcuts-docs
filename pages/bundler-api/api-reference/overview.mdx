## Endpoints

Currently, the Bundle API has only one endpoint

    - [POST /v1/shortcuts/bundle](/bundler-api/api-reference/bundle) - Bundle a list of actions into a single tx. Returns a transaction that can be submitted to an Ethereum node.

## Actions

The bundle endpoint accepts a list of actions. Users can fetch all the available actions in this endpoint from the [Metadata API](/bundler-api/introduction).

    - GET /v1/shortcuts/actions
