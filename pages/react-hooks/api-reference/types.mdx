# Types

### Token
```ts
type Token = {
  address: string;
  name: string;
  symbol: string;
  decimals: number;
  chain: number;
  project: string;
  protocol: string;
  poolAddress: string;
  primaryAddress: string;
  underlyingPoolTokens: string[];
}
```

#### Route
Describes a route that was found via the `useExecutePosition` hook.

```ts
type Route = {
  amountOut: string;
  createdAt: number;
  gas: number;
  route: Hop[];
  tx: Transaction;
}
```

#### Hop
Describes a hop taken for a specific route

```ts
type Hop = {
  tokenIn: string[];
  tokenOut: string;
  positionOutId: string;
  protocol: string;
  action: string;
  primary: string;
}
```
