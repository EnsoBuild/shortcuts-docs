## Endpoints

Learn more about the different endpoints of the multichain API:

    - `GET /api/experimental/multichain/shortcut/route`  Get an easy-to-submit bytecode to go from a token on another chain, including LPs and DeFi positions. Returns a transaction that can be submitted to an Ethereum node.

## Authentication

Enso authenticates your API requests using your account’s API keys. Once you’re set up, ensure that you specify your key with the 0x-api-key header parameter in your requests.

You can create, access or revoke your API keys via the Enso portal.
